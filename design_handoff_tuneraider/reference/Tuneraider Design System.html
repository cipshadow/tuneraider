01 COLOR

Five greens, darkest to lightest — the classic 8-bit handheld ramp. Background uses the deepest shade; brightness climbs toward foreground text and values. Red is the only off-ramp hue, reserved strictly for the wrong answer.

02 TYPOGRAPHY

One face does everything: Press Start 2P, the bitmap pixel font. Single weight (400), no italics. All-caps for titles and labels; sentence case stays readable in body copy. Generous letter-spacing keeps the pixels from fusing.

FX BURST — WEPA / NOOO STAMP (LIVE)

Overlapping retro titles pop in back-to-front and cascade upward, each ringed in a dark outline so the tight stack stays legible. A nailed answer fires WEPA in joyful colors with a white flash; a miss fires NOOO in danger reds with a red flash. Reusable — fireCorrect('WEPA') / fireWrong('NOOO') take any word.

6 layers · 15px rise + 7px drift per echo (tight stack) · 3px dark outline · back pops first, front stamps last (45ms stagger) · ~2s total.
